Class: Problem

Represents an instance of a problem pertaining to conformance to OBO guidelines

  • NOTE: this is an abstract class and should not be instantiated directly

URI: semsql_obo:Problem

classDiagram Problem <|-- LexicalProblem Problem <|-- TrailingWhitespaceProblem Problem <|-- PropertyUsedWithDatatypeValuesAndObjects Problem <|-- NodeWithTwoLabelsProblem Problem <|-- AllProblems Problem : predicate Problem : subject Problem : value

SELECT * FROM problem;

Slots

Name Cardinality and Range Description

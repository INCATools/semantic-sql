Class: RdfFirstStatement

A statement that connects a list to its first element. This is a low-level triple, it is unlikely you need to use this directly. It is used to define rdf_list_member_statement, which is more useful

URI: semsql_rdf:RdfFirstStatement

classDiagram RdfListStatement <|-- RdfFirstStatement RdfFirstStatement : datatype RdfFirstStatement : language RdfFirstStatement : object RdfFirstStatement : predicate RdfFirstStatement : stanza RdfFirstStatement : subject RdfFirstStatement : value
